## 2. 产品概述

### 2.1 产品背景
心理健康问题日益受到社会关注，但专业心理咨询资源有限。本产品旨在通过 AI 技术为用户提供便捷的心理健康服务，包括情绪记录、AI 咨询、知识科普等功能，帮助用户更好地了解和管理自己的心理状态。

### 2.2 产品定位
- **目标用户**：关注心理健康的普通用户、学生群体、职场人士
- **核心价值**：7x24 小时 AI 心理陪伴 + 情绪数据追踪 + 专业知识学习
- **差异化优势**：智能情绪分析、个性化建议、隐私保护

### 2.3 目标用户
| 用户类型 | 特征 | 核心需求 |
|----------|------|----------|
| 普通用户 | 有轻度情绪困扰，希望自我调节 | 情绪记录、AI倾诉、知识学习 |
| 学生群体 | 学业压力大，需要心理支持 | 压力管理、考试焦虑缓解 |
| 职场人士 | 工作压力大，时间碎片化 | 快速情绪释放、高效心理调节 |
| 管理员 | 平台运营人员 | 内容管理、用户数据分析 |

## 4. 页面详细设计

### 4.1 用户端

#### 4.1.1 官网首页 (index.html)
**页面目的**：品牌展示、功能介绍、引导注册

**页面结构**：
- Hero区域：品牌标语、CTA按钮
- 功能特性：4大核心功能卡片展示
- 统计数据：用户量、咨询次数等数据展示
- 底部导航：快速链接

**核心交互**：
- 统计数据数字滚动动画
- 功能卡片hover效果
- CTA按钮跳转登录/注册

---

#### 4.1.2 登录页 (login.html)
**页面目的**：用户身份认证

**页面结构**：
- 左侧：品牌图片/宣传语
- 右侧：登录表单
  - 用户名输入框
  - 密码输入框
  - 登录按钮
  - 注册链接

**核心交互**：
- 表单验证（非空校验）
- 登录成功：
  - 普通用户 → 首页
  - 管理员(admin/admin) → 管理端首页
- 错误提示（账号或密码错误）
- LocalStorage 保存登录状态

---

#### 4.1.3 注册页 (register.html)
**页面目的**：新用户账号创建

**页面结构**：
- 左侧：品牌宣传
- 右侧：注册表单
  - 用户名
  - 邮箱
  - 密码
  - 确认密码
  - 注册按钮

**核心交互**：
- 实时表单验证
- 密码强度提示
- 注册成功跳转登录页

---

#### 4.1.4 AI咨询页 (chat.html)
**页面目的**：提供 AI 心理咨询对话服务

**页面结构**：
- **左侧边栏**：
  - 用户头像和状态
  - 情绪状态卡片（情绪评分、今日感受）
  - AI建议卡片
  - 会话历史列表
- **主聊天区**：
  - 顶部导航
  - AI助手信息栏
  - 聊天内容区
  - 消息输入框

**核心交互**：
- 发送消息（Enter发送，Shift+Enter换行）
- 快捷提示点击发送
- 字数统计（0/500）
- 新建对话功能
- 模拟AI自动回复
- 会话历史切换

**数据结构**：
```javascript
{
  sessionId: string,
  messages: [
    { role: 'user'|'ai', content: string, timestamp: Date }
  ],
  createTime: Date
}
```

---

#### 4.1.5 知识库列表页 (knowledge.html)
**页面目的**：展示心理健康知识文章

**页面结构**：
- **左侧边栏**：推荐阅读列表（5篇）
- **右侧主内容**：
  - 搜索框 + 分类标签
  - 文章卡片网格
  - 分页器

**核心交互**：
- 分类筛选（全部/压力管理/睡眠健康/情绪调节/人际关系/个人成长）
- 搜索功能
- 文章卡片点击进入详情
- 分页切换

**文章卡片结构**：
- 封面图
- 分类标签
- 标题
- 摘要（2行截断）
- 发布日期 + 阅读量

---

#### 4.1.6 文章详情页 (article-detail.html)
**页面目的**：展示完整文章内容

**页面结构**：
- **左侧边栏**：推荐阅读（同列表页）
- **右侧主内容**：
  - 面包屑导航
  - 文章元信息（分类、日期、阅读量）
  - 文章标题
  - 摘要提示框
  - 作者信息
  - 正文内容
  - 相关标签

**核心交互**：
- 返回列表按钮
- 标签点击筛选

---

#### 4.1.7 情绪日记页 (diary.html)
**页面目的**：记录每日情绪状态

**页面结构**：
- **情绪评分卡片**：
  - 10星评分系统
  - 情绪标签（开心、满足、兴奋、平静、焦虑、压力、疲惫、沮丧、愤怒、失落、孤独、困惑）
- **详细记录卡片**：
  - 触发因素输入
  - 此刻想法文本框
  - 睡眠质量选择（1-5分）
  - 压力水平选择（1-5分）
  - 保存/清空按钮

**核心交互**：
- 星级评分点击
- 多选情绪标签
- 表单验证
- 本地存储保存
- 成功提示

**数据结构**：
```javascript
{
  id: string,
  rating: number,      // 1-10
  emotions: string[],  // 选中的情绪标签
  trigger: string,     // 触发因素
  thoughts: string,    // 想法记录
  sleep: number,       // 1-5
  stress: number,      // 1-5
  date: Date
}
```

## 6. 交互设计

### 6.1 全局交互

**登录状态管理**：
- 使用 LocalStorage 存储 `isAdminLoggedIn` / `isUserLoggedIn`
- 管理端页面自动检查登录状态，未登录跳转登录页
- 退出登录清除存储并跳转首页

**导航交互**：
- 当前页面高亮显示
- 页面切换平滑过渡

**弹窗交互**：
- 点击遮罩层关闭
- ESC 键关闭
- 关闭按钮
- 表单提交后自动关闭

### 6.2 表单验证

| 字段 | 验证规则 | 错误提示 |
|------|----------|----------|
| 用户名 | 必填，3-20字符 | 请输入用户名 |
| 密码 | 必填，6-20字符 | 请输入密码 |
| 邮箱 | 必填，邮箱格式 | 请输入有效邮箱 |
| 标题 | 必填 | 请输入标题 |
| 评分 | 必选 | 请选择评分 |

### 6.3 响应式断点

| 断点 | 宽度 | 适配 |
|------|------|------|
| 大屏 | ≥1440px | 完整布局 |
| 中屏 | 1024-1439px | 隐藏侧边栏 |
| 小屏 | 768-1023px | 简化导航 |
| 移动端 | <768px | 单列布局 |
